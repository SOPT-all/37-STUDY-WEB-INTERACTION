## 🔀 PR & 브랜치 규칙

### 🔸 브랜치명
- 형식: `week{n}-{이름}`  
- 예시: `week1-chaeyoung`, `week3-yoonji`  
- 각 주차별 실습은 자신의 브랜치에서 진행합니다.

### 🔸 PR 제목
- 형식: `[Week {n}] {이름}`  
- 예시: `[Week 2] 어진`

### 🔸 PR 내용
- 학습 내용 요약  
- 실습 파일 설명  
- 느낀 점 및 피드백 요청 사항 포함

### 🔸 리뷰 규칙
- 최소 2명 이상에게 간단한 피드백 댓글 작성
